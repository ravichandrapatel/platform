# Terraform Refresh Daily (sample workflow)

This workflow is a **sample/testing workflow** used to validate the Terraform refresh behavior described in the platform docs.

- It is **not intended to be released** as a reusable workflow.
- Keep it under `workflows/terraform-refresh-daily/` for reference and for local/CI testing.

If you need a real daily refresh in another repo, copy/adapt this file into that repo’s `.github/workflows/`.
