# Changelog – platform actions and images

All notable changes to platform actions, workflows, and container images are documented here.

- **Added** – New actions, workflows, or images.
- **Changed** – Updates to existing components (inputs, behavior, dependencies).
- **Fixed** – Bug fixes or security fixes.
- **Removed** – Deprecated or removed components.
